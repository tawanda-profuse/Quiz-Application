<h2>Installation Steps:</h2>

<p>Type the following commands after extracting the Zip file.</p>
<ol>
  <li>npm install</li>
  <li>npm init</li>
  <li>node app</li>
  <li>Lastly, open your browser at the port specified in app.js (should be port 3000)</li>
</ol>
